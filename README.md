# Mik3Bot & BotTester
> - author: Mike Finch
> - date: March 2026

## Mik3Bot
Is a self contained app, that calls a LLM locally

## BotTester
Allows a simple prompt to be entered, it then uses a Requestor Bot to generate a better query based off a configurable persona. The Requestor Bot then makes a call to the Responder Bot, which uses a different model to respond, again this uses the characteristics defined in a configurable proforma to generate a response. Finally a Reasoning Bot determines if Responder Bot has accurately answered the Request generated by Requestor Bot
